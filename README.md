# Swiggy Data Analysis using Web Scraping and EDA

This project collects Swiggy food delivery data using web scraping and performs Exploratory Data Analysis.

## Tools Used
Python
BeautifulSoup
Pandas
Matplotlib
Seaborn

## Files
swiggyWebscraproj.ipynb – Web scraping and analysis code  
swiggy_food_delivery_analysis.csv – Dataset  
swiggy_project_presentation_A1A.pdf – Project presentation

## Steps
1. Web scraping
2. Data cleaning
3. Exploratory Data Analysis
4. Data visualization
